Machine Learning Thesis

Hyperspherical Quantization: A Geometry-Aware Deep Quantization Framework (Advised by: M. I. AlHajri)

I work on optimizing quantization for deep neural networks using a geometry-aware approach. I mainly focus on using non-Euclidean geometric structures (hyperspherical manifolds) to encode spatial information into the quantization scheme.
